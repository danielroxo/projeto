Designing a new feature

The Challenge

Our goal was to design a new feature on an already existing global app.

It was intended for our feature to be intricate enough, while still sticking to the existing brand design system.

The Solution

The new Karaoke feature will allow users to read the lyrics and sing their favorite songs solo or in group, via desktop or mobile. We had one month to develop this project.
